# Casio fx-991ES Scientific Calculator Simulator

An open-source scientific calculator clone inspired by the layout, styling, and functionality of the classic **Casio fx-991ES** handheld device. Built cleanly with Python and Tkinter UI scripting.

## Features
- Standard Number Pad & Operations (`+`, `-`, `×`, `÷`)
- Scientific Modules: `sin`, `cos`, `tan` (computed in degrees)
- Power indices (`x²`, `x³`, `xⁿ`) & Square roots (`√`)
- Natural Display emulation engine using custom fonts.
